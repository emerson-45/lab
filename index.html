E1
¿Bajo qué condiciones son verdaderas p y q?
30 %
[ (q ↔ p) ∧ p ] → (p ∧ ¬q)
PASO 1 — Colocar en orden todas las sub-expresiones
Identificamos cada sub-expresión de la proposición compuesta, de menor a mayor complejidad, como se hace en álgebra con cada término:

Sub-expresiones en orden

  • 1. p
  • 2. q
  • 3. ¬q
  • 4. q ↔ p
  • 5. (q ↔ p) ∧ p
  • 6. p ∧ ¬q
  • 7. [ (q ↔ p) ∧ p ] → (p ∧ ¬q) ← expresión final
PASO 2 — Número de filas
n = número de proposiciones simples = 2 (p y q)
Número de filas = 2n = 22 = 4 filas
PASO 3 — Construir la tabla de verdad
Reglas usadas:
• p ↔ q: V solo cuando ambas tienen el mismo valor de verdad.
• p ∧ q: V solo cuando ambas son V.
• p → q: F solo cuando antecedente=V y consecuente=F.
• ¬q: invierte el valor de q.
p q ¬q q ↔ p (q↔p) ∧ p p ∧ ¬q [(q↔p)∧p] → (p∧¬q)
VV F V V F F
VF V F F V V
FV F F F F V
FF V V F F V
CLASIFICACIÓN
✦ Contingencia (hay V y F)

✅ Condiciones para que la proposición sea VERDADERA

  • Fila 2: p = V, q = F
  • Fila 3: p = F, q = V
  • Fila 4: p = F, q = F

⚠ La proposición es FALSA únicamente cuando

Fila 1: p = V y q = V

Explicación: el antecedente (q↔p)∧p = V∧V = V, y el consecuente p∧¬q = V∧F = F.
Como V → F = F, la condicional falla solo en ese caso.

Valor de expresiones booleanas por sustitución
30 %
DATOS
x₁ = 0  |  x₂ = 1  |  x₃ = 0
Expresión a)   x̄₁ ∨ ( x̄₂ ∧ x₃ )
x̄₁ ∨ ( x̄₂ ∧ x₃ )
PASO 1 — Calcular los complementos (NOT)
x₁ = 0  →  x̄₁ = ¬(0) =1
x₂ = 1  →  x̄₂ = ¬(1) =0
x₃ = 0  →  sin cambio =0
PASO 2 — Resolver el paréntesis interior (AND primero)
x̄₂ ∧ x₃  =  0 ∧ 0  = 0  (AND: F solo cuando hay al menos un 0)
PASO 3 — Resolver el OR final
x̄₁ ∨ ( x̄₂ ∧ x₃ )  =  1 ∨ 0  = 1  (OR: V cuando al menos uno es 1)
✔ Resultado: x̄₁ ∨ (x̄₂ ∧ x₃) = 1 (VERDADERO)
Expresión b)   ( x₁ ∧ x̄₂ ) ∧ (x₁ ∨ x̄₃)
( x₁ ∧ x̄₂ ) ∧ ‾(x₁ ∨ x̄₃)
PASO 1 — Calcular los complementos (NOT)
x₁ = 0  →  sin complemento (se usa directo) =0
x₂ = 1  →  x̄₂ = ¬(1) =0
x₃ = 0  →  x̄₃ = ¬(0) =1
PASO 2 — Resolver el primer paréntesis (AND)
x₁ ∧ x̄₂  =  0 ∧ 0  = 0  (AND: ambos deben ser 1 para dar 1)
PASO 3 — Resolver el interior del segundo paréntesis (OR)
x₁ ∨ x̄₃  =  0 ∨ 1  = 1  (OR: al menos un 1, resultado 1)
PASO 4 — Aplicar la barra de negación sobre todo el paréntesis: ‾(x̄₁ ∨ x̄₃)
‾(x₁ ∨ x̄₃)  =  ¬(1)  = 0  (NOT aplicado al resultado completo del OR)
PASO 5 — Resolver el AND final
(x₁∧x̄₂) ∧ ‾(x₁∨x̄₃)  =  0 ∧ 0  = 0  (AND: ambos operandos son 0, resultado 0)
✖ Resultado: (x₁∧x̄₂) ∧ ‾(x₁∨x̄₃) = 0 (FALSO)

PASO 1 — Listar todas las sub-expresiones booleanas

Sub-expresiones en orden de construcción

  • ① x₁  → entrada directa
  • ② x₂  → entrada directa
  • ③ x₃  → entrada directa
  • ④ x̄₂  → compuerta NOT aplicada a x₂
  • ⑤ x̄₃  → compuerta NOT aplicada a x₃
  • ⑥ x₁ ∧ x̄₂  → compuerta AND con x₁ y x̄₂
  • ⑦ x₁ ⊕ x̄₃  → compuerta XOR con x₁ y x̄₃
  • ⑧ ‾(x₁ ⊕ x̄₃)  → compuerta NOT aplicada al resultado del XOR (barra larga)
  • ⑨ (x₁ ∧ x̄₂) ∧ ‾(x₁ ⊕ x̄₃)  → compuerta AND final
NOTA SOBRE XOR NEGADO: ‾(x₁ ⊕ x̄₃)
La barra larga encima del XOR lo niega: primero se calcula el XOR, luego se aplica NOT al resultado. Esto equivale a una compuerta XNOR, que da 1 cuando ambas entradas son iguales.
